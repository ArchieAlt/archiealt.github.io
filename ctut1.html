What is C ?

C is a general-purpose computer programming language. It was created in the 1970s by Dennis Ritchie, and remains very widely used and influential. By design, C's features cleanly reflect the capabilities of the targeted CPUs. It has found lasting use in operating systems, device drivers, and protocol stacks, but its use in application software has been decreasing.[7] C is commonly used on computer architectures that range from the largest supercomputers to the smallest microcontrollers and embedded systems. (Source : Wikipedia)

CodeBlocks

Code::Blocks is a free C/C++ and Fortran IDE built to meet the most demanding needs of its users. It is designed to be very extensible and fully configurable.Built around a plugin framework, Code::Blocks can be extended with plugins. Any kind of functionality can be added by installing/coding a plugin. For instance, event compiling and debugging functionality is provided by plugins!

Installing CodeBlocks

First of all we are going to install the IDE , CodeBlocks.It's very simple so let's do it !

Go to the CodeBlocks Official Site and go to their download page. Go to Binary or Nigthly releases (depending if you want recently updated) and select CodeBlocks-setup.exe

I have tried to download it with the MinGW compiler included but it couldn't load the compiler.

Installing MinGW

You are going to install MinGW installation manager, go to this SourceForge page after you run the exe just follow the instructions !

Once you are in the MingGW IM(installation manager) you are going to met with a screen like this

Select all the packages by clicking Right Mouse Button and selecting : "Mark For Installation" button. Once you have selected the packages then you will click Installation button and click update catalouge.Then just close it when it says it's finished.

And you're done ! Once you run CodeBlocks it should just load the mingw compiler , if it doesn't still work go into CodeBlocks , press settings and select compiler.Then select MinGW !
