Today I will show you how to make your first C program ! So let's get started !

First C program

Open your IDE (probably CodeBlocks since I showed you how to download and set it up in the last tutorial !)

Make a new project by pressing Create a new project button , go to files , select C source file type,and set up where you want to save your files !

Programming your first program

Now you are presented with your IDE so let's start !

First of all you are going to include the stdio.h library , stdio stands for standard input-output.

#include <stdio.h> // This is a comment

Now you will need to declare the Main function. The main function is where your code runs !

			
#include <stdio.h> // This is a comment
		
int main(){

return 0;
}

What I did here is declared the main funtion , the code you write goes into the curly braces and before the "return 0;" return 0 basically means that the program executed succesfully ! Now just add the printf function , by typing: 

printf("your text here");
, i.e. entering your text inbetween the quotes and finishing the function with a semicolon(;) !

#include <stdio.h> // This is a comment
		
int main(){
	printf("Hello World !");
return 0;
}

This is how the finished program looks like ! On codeblocks you run it by clicking the green button ! If you haven't built your code press the green run button with the gear ! And a Command Prompt window should pop up with the text you inputed ! That's it you wrote your first program good job !
